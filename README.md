🏠 MEGAN-IoT-Home-Automation
MEGAN is an IoT-based home automation system that enables users to control and monitor household appliances such as lights, fans, lamps, and sockets remotely through a mobile app, using real-time database integration and voice commands for a smart and efficient living experience.

🚀 Features
- 🔌 Remote control of home appliances
- 📱 Mobile app integration
- 📊 Real-time monitoring using IoT
- 🎤 Voice command support
- ⚡ Energy-efficient automation
- 🔒 Secure and reliable system
  
 🛠️ Tech Stack
- Arduino / Microcontroller
- IoT Modules (ESP8266 / ESP32)
- Relay Modules
- Mobile App Interface
- Firebase - Cloud / Real-time Database
